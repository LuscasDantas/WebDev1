Conceitos Gerais

<article> O elemento article representa uma composição independente em um documento, página, aplicação, ou site, ou que é destinado a ser distribuido de forma independente ou reutilizável
<aside> O elemento define algum conteúdo além daquele em que está colocado (como uma barra lateral). O conteúdo deve estar indiretamente relacionado ao conteúdo circundante.
<details> O elemento especifica detalhes adicionais que o usuário pode abrir e fechar sob demanda. A tag é frequentemente usada para criar um widget interativo que o usuário pode abrir e fechar.
<figcaption> O elemento define uma legenda para um elemento <figure>. Pode ser colocado como o primeiro ou último filho do elemento
<figure> O elemento especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de códigos, etc. Embora o conteúdo do elemento esteja relacionado ao fluxo principal, sua posição é independente do fluxo principal e, se removido, não deverá afetar o fluxo do documento.
<footer> O elemento define um rodapé para um documento ou seção. Um <footer> normalmente contém:
  • Informações de autoria
  • Informações sobre direitos autorais
  • Informações de contato
  • Mapa do site
  • Voltar ao topo links
  • Documentos relacionados
<header> O elemento representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação. Um <header> normalmente contém:
  • um ou mais elementos de título (<h1> - <h6)
  • logotipo ou ícone
  • informações de autoria
<main> O elemento especifica o conteúdo principal de um documento. O conteúdo dentro do <main> deve ser exclusivo do documento. Ele não deve conter nenhum conteúdo repetido em documentos, como barras laterais, links de navegação, informações de direitos autorais, logotipos de sites e formulários de pesquisa.
<mark> O elemento define o texto que deve ser marcado ou destacado.
<nav> O elemento define um conjunto de links de navegação. Observe que nem todos os links de um documento devem estar dentro de um <nav>. O
destina-se apenas a grandes blocos de links de navegação. Navegadores, como leitores de tela para usuários com deficiência, podem usar esse elemento para determinar se devem omitir a renderização inicial desse conteúdo
<section> O elemento define uma seção em um documento.
<summary> O elemento define um cabeçalho visível para o elemento <details>. O título pode ser clicado para visualizar/ocultar os detalhes.
<time> define uma hora específica (ou data e hora). O datetime atributo deste elemento é usado para traduzir a hora em um formato legível por máquina para que os navegadores possam oferecer a adição de lembretes de data por meio do calendário do usuário, e os mecanismos de pesquisa possam produzir resultados de pesquisa mais inteligentes.
