Códigos fontes e afins

<kbd> O elemento é usado para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
<samp> O elemento é usado para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
<code> O elemento é usado para definir um trecho de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.
<var> O elemento é usado para definir uma variável em programação ou em uma expressão matemática. O conteúdo interno normalmente é exibido em itálico.
<pre> A tag define texto pré-formatado. O texto em um <pre> é exibido em uma fonte de largura fixa e preserva espaços e quebras de linha. O texto será exibido exatamente como está escrito no código-fonte HTML.
