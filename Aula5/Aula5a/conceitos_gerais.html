Conceitos Gerais

O que é semântica?

A semântica é a área da linguística que estuda o significado e a sua relação com o significante. O significado está associado ao sentido e, portanto, ao conteúdo e ao contexto; o significante está associado à forma (de palavras ou de sinais, de grafia ou de som).

O que é semântica HTML (semântica no contexto da marcação HTML)?

Semântica HTML refere-se ao uso de tags que transmitem exatamente o significado do conteúdo que está contido nelas. Um elemento semântico descreve claramente seu significado tanto para o navegador quanto para o desenvolvedor. Fornecendo informações adicionais que ajudam a definir a importância de cada part da sua página.

O que são elementos semânticos HTML?

Os elementos semânticos do HTML (tags semânticas) são aqueles que definem o sentido do conteúdo neles inserido

Exemplos de elementos semânticos

Podemos citar como elementos semânticos:

  • <figcaption>

    O elemento HTML figcaption representa uma legenda associada com uma figura ou ilustração descrita pelo resto dos dados do elemento <figure> que é seu elemento pai.

  • <main>

    O elemento <main> define o conteúdo principal dentro do <body> em seu documento ou aplicação. Entende-se como conteúdo principal aquele relacionado diretamente com o tópico central da página ou com a funcionalidade central da aplicação.

Os elementos não semânticos do HTML mais comuns são:

  • <div>

    O elemento <div> indica uma separação/divisão no conteúdo da página. No entanto, ele não define o que será inserido na tag.

  • <span>

    O elemento <span> é um container generico em linha para conteúdo fraseado, que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo, ou para compartilhar valores de atributos.

Benefícios da utilização de elementos semânticos

Os elementos semânticos permitem que o conteúdo da página seja organizado e acessível. Além de torná-las otimizadas para mecanismos de busca.
A utilização de elementos semânticos corretos permite que os navegadores, leitores de tela e outros dispositivos possam interpretar corretamente o conteúdo de uma página, tornando-a mais acessível para pessoas com deficiências visuais ou cognitivas
